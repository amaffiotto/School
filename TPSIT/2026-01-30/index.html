Variabili

Esempi di dichiarazione di variabili:


let var1 = 12;      // Numero
let var2 = "12";    // Stringa
let var3;           // Undefined
const pigreco = 3.14; // Costante
var parola;         // Vecchia sintassi

Tipi di dato

  • number
  • string
  • boolean
  • null
  • undefined

Stringhe e Template Literal

Le stringhe possono essere delimitate da:

  • " " Doppi apici
  • ' ' Singoli apici
  • ` ` Backtick (Template Literal)

I Backtick permettono di inserire variabili direttamente nella stringa con ${...}. 


let nome = "Luca";
let saluto = `Ciao ${nome}!`; // "Ciao Luca!"

// Multilinea
let testo = `Questo è un testo
su più righe`;

// Metodi utili (Case sensitive)
console.log(nome.length);   // Lunghezza stringa
console.log(nome.charAt(0)); // Primo carattere ('L')
console.log(nome.indexOf('L')); // Indice di 'L' (0)
console.log(nome.lastIndexOf('a')); // Ultimo indice di 'a'
console.log(nome.indexOf('z')); // -1 (Non trovato)

// substring(inizio, fine_esclusa)
// Se ometto la fine, va fino in fondo
console.log(nome.substring(0, 3)); // "Luc" (da 0 a 3 escluso)
console.log(nome.substring(1));    // "uca" (da 1 alla fine)

// Maiuscolo / Minuscolo
console.log(nome.toUpperCase()); // "LUCA"
console.log(nome.toLowerCase()); // "luca"

// Trim (rimuove spazi inizio/fine)
let spazi = "  Ciao  ";
console.log(spazi.trim());       // "Ciao"

Split e Join

Trasformare Stringhe in Array e viceversa.


let frase = "uno,due,tre";

// SPLIT: Stringa -> Array
let splittati = frase.split(","); 
console.log(splittati); // ["uno", "due", "tre"]

// JOIN: Array -> Stringa
let uniti = splittati.join("-");
console.log(uniti); // "uno-due-tre"

Concatenazione Stringhe

Unire due o più stringhe in una sola.


let n = "Ciao";
let m = "Mondo";

// Operatore +
let saluto = n + " " + m; 
console.log(saluto); // "Ciao Mondo"

// concat()
let saluto2 = n.concat(" ", m);
console.log(saluto2); // "Ciao Mondo"

Scope delle Variabili

Lo scope determina l'accessibilità (visibilità) delle variabili.

  • Block Scope (let, const): La variabile esiste solo all'interno del blocco {}.
  • Function Scope (var): La variabile è visibile in tutta la funzione.
  • Global Scope: Visibile ovunque nello script.

{
    let blocco = "Segreto"; // Invisibile fuori
    var funzione = "Pubblico"; // Visibile fuori (se non siamo in una func)
}
// console.log(blocco); // Error
console.log(funzione); // "Pubblico"

Operatori

Aritmetici

  • + Addizione
  • - Sottrazione
  • * Moltiplicazione
  • / Divisione
  • % Modulo (Resto)

let somma = 10 + 5; // 15
let resto = 10 % 3; // 1

Confronto

== confronta solo il valore

=== uguaglianza stretta (valore e tipo)


let v1 = 5;
let v2 = "5";

v1 == v2  // true
v1 === v2 // false

Logici

  • && AND: Vero se entrambi veri
  • || OR: Vero se almeno uno vero
  • ! NOT: Inverte il risultato

let x = true;
let y = false;

x && y // false
x || y // true
!x     // false

Funzioni


function somma(a, b) {
    return a + b;

let risultato = function() {

Arrow Functions


// Sintassi concisa
const somma = (a, b) => a + b;

// Con blocco di codice
const saluta = () => {
    console.log("Ciao Mondo!");
