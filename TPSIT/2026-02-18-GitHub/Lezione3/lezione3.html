Questo usa un font personalizzato da Google Fonts!

5. CSS Custom Properties - Variabili CSS

Le variabili CSS permettono di riutilizzare valori e creare temi dinamici!

Sintassi Base

/* Definizione */
:root {
    --colore-primario: #3498db;
    --padding-base: 20px;
    --font-size-grande: 24px;
}

/* Utilizzo */
.box {
    color: var(--colore-primario);
    padding: var(--padding-base);
    font-size: var(--font-size-grande);
}
